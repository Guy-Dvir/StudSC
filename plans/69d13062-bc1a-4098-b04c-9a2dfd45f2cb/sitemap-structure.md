## Information Architecture
*   **Home** (Brand USP, Bestsellers, Social Proof)
*   **Shop All**
    *   **Dog Food** (Puppy, Adult, Senior, Grain-Free)
    *   **Cat Food** (Kitten, Adult, Indoor)
    *   **Treats & Toppers**
    *   **Supplements & Care**
*   **The Peteat Method** (Sourcing, Quality Standards, Raw vs. Kibble)
*   **Feeding Guide** (Interactive Portion Calculator)
*   **Customer Support** (FAQs, Shipping, Returns, Contact)
*   **My Account** (Subscription Portal, Order History, Loyalty Points)

## Main Navigation Structure
*   **Primary Navigation:** Desktop Mega Menu featuring visual icons for Dog/Cat categories; "Personalized Plan" CTA highlighted in a secondary color.
*   **Utility Navigation:** Sticky header containing Search, Account Login, and a Slide-out Cart with a "Free Shipping" progress bar.
*   **Footer:** Newsletter signup, trust badges (ISO/HACCP), and deep links to specific protein-based diets (e.g., "Beef for Dogs").

## Key User Flows
*   **The New Parent Flow:** Home -> Feeding Calculator -> Input Pet Profile -> Recommended Personalized Meal Plan -> Subscription Checkout.
*   **The Efficiency Flow:** Search/Nav -> Collection Page (Filtered by Protein) -> Quick-Add to Cart -> Express Checkout.
*   **The Educator Flow:** Blog/Social -> "The Peteat Method" -> Product Page (Ingredient Transparency) -> Purchase.

## Page-Level Notes
*   **Collection Pages:** Implement "Shop by Need" filters (e.g., Joint Support, Digestion) alongside traditional protein filters.
*   **Product Detail Pages (PDP):** Prioritize a "Subscribe & Save" toggle over one-time purchases; include accordion sections for full ingredient breakdowns and caloric analysis.
*   **Feeding Calculator:** A lead-capture tool that provides value before the sale, ensuring customers buy the correct volume for their pet’s weight and activity level.
